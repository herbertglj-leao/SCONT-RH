Segurança e criptografia local

Estrutura preparada para senha mestra e Web Crypto API
Em front-end puro, a proteção real depende de criptografia local, redução de exposição de senha e armazenamento seguro no navegador.

Configuração de senha mestra

Recomendações de segurança

  • Não armazenar senha em texto puro.
  • Usar PBKDF2 + AES-GCM.
  • Derivar chave a partir de senha mestra.
  • Aplicar timeout de sessão.
  • Limpar memória quando possível.
  • Planejar migração futura para back-end seguro.
