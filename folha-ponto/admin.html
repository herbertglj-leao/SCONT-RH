Cadastrar Nova Regra

Tags Disponíveis:

Use estas tags exatamente como estão escritas (com as chaves) para criar os padrões.

  • {CODIGO_EMPRESA} : Extrai o código (DE) ou insere o código (PARA).
  • {NOME_EMPRESA} : Insere o nome da empresa (apenas no PARA).
  • {NOME_ARQUIVO} : Extrai o nome original (DE) ou insere o nome (PARA).
  • {MM} : Mês da competência (2 dígitos).
  • {AAAA} : Ano da competência (4 dígitos).
  • {IGNORAR} : Ignora uma parte do texto original (apenas no DE).

Exemplo DE: {CODIGO_EMPRESA}_{NOME_ARQUIVO}_{IGNORAR}_{MM}{AAAA}

Exemplo PARA: {NOME_ARQUIVO}_{NOME_EMPRESA}_{AAAA}_{MM}

Instruções

Sua planilha deve conter as seguintes abas:

  • Aba "Empresas": Coluna A = Código da Empresa, Coluna B = Nome da Empresa
  • Aba "Empregados": Coluna A = Código da Empresa, Coluna B = Código do Empregado, Coluna C = Nome do Empregado
  • Aba "Rubricas": Coluna A = Código da Empresa, Coluna B = Código da Rubrica, Coluna C = Nome do Evento (ex: horasTrabalhadas, horasExtras50, horasExtras100, horasNoturnaConvertida, horasDevidas)

⚠️ Aviso: A importação irá substituir todos os dados existentes (Empresas, Empregados e Rubricas)!
