Segurança e criptografia local

Proteção de senhas e dados sensíveis em front-end
Em front-end puro, a proteção real depende de criptografia local com Web Crypto API. Senhas são preparadas para PBKDF2 + AES-GCM, mas a implementação completa requer back-end seguro para armazenamento.

Configuração de Senha Mestra

Recomendações de Segurança

  • Não armazenar senha em texto puro.
  • Usar PBKDF2 + AES-GCM para criptografia.
  • Derivar chave a partir de senha mestra.
  • Aplicar timeout de sessão.
  • Limpar memória quando possível.
  • Planejar migração futura para back-end seguro.
